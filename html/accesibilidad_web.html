La accesibilidad web consiste en una interfaz de usuario inclusiva, accesible a las personas con alguna discapacidad, ya sea visual, social, u otra, para que así pueda navegar por el sitio de la mejor forma posible.


Principios de la accesibilidad web

Perceptibilidad

El contenido de la página debería ser perceptible a los usuarios, tanto de forma visual como auditiva. El mejor ejemplo de perceptibilidad es el uso de íconos, etiquetas y descripciones de las imágenes dentro de un sitio web

La página Udemy, utiliza imágenes para mostrar sus respectivas categorías de cursos y demás, haciéndolo apto para personas analfabetas o daltónicas.


Operabilidad

Este principio es respecto a la manipulación que tiene el usuario sobre la página. Lo ideal es que el visitante pueda ver el contenido del sitio y desplazarse por el mismo a su gusto, sin depender de un tiempo establecido para apreciar la información que se ofrece. Un ejemplo sobre una mala operabilidad sería una página en la que el contenido se pasa como si fuera un video, dificultado así la lectura para aquellas personas que leen lento o que tiene dislexia.

Amazon usa un carrusel en su sitio, al inicio de la página, pero este no se recorre de forma automática, permitiendo así que el usuario pueda estar en cada diapositiva el tiempo que desee.


Robustez

Indica que el sitio web debe funcionar, sin importar el dispositivo o navegador en el que se vea, y al mismo tiempo tener compatibilidad con las herramientas hechas para ampliar la accesibilidad de un usuario especifico.

Google talkback y voice access son softwares de accesibilidad, así que un bueno ejemplo sería usar estos programas en la página web y así corroborar que cumple con este principio. De igual forma es posible ver el sitio en diferentes navegadores, buscando puntos de mejora con respecto a la inclusividad digital.


Las personas con una discapacidad visual, cognitiva, auditiva y las personas con analfabetismo son algunos de los grupos de usuarios por los que es necesario la accesibilidad web. Un sitio hecho de este modo ayuda y demuestra un apoyo a la inclusividad web.

Actualmente las páginas web son fundamentales para el conocimiento y la comercialización de las empresas, abarcando aspectos en la educación, gobierno, salud, entre otros. Es por esta razón por la que se debe procurar la creación de sitios web accesible a cualquier usuario, ofreciendo una igualdad de acceso y eliminando las barreras de exclusividad.
